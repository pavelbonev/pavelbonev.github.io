Games Info

First screen provides info about all games in a tournament. The games could be filtered by typing text in the search box located in the upper left corner. By choosing any passed tournament the user could see info about it

Groups Info

Taping on group letter button opens Groups Screen where the standings and qualified teams are shown. The standings and qualified teams are updated after every change of the game result .

Giving predictions

By taping on the prediction result button the user opens the prediction screen. The user could give and change their prediction for a game until the game starts.

Users' standings

By taping on the standings button User standings screen is opened. The standings are formed by the users' predictions and games results. If the game result is the same as the prediction, the user is awarded with three points. If the user guessed just the game outcome and not the exact result, the user is awarded by one point. If points are the same between two or more users, the number of guessed outcomes is taken into account. If the outcomes are the same between those users - the number of predictions matters, and "less is more".

Swarms

The swarms are premium feature which costs 1$ per year without taxes. This feature makes possible grouping people in swarms and filtering standings per swarm. It is accessible by clicking button SWARMS on main screen. When user creates a swarm he/she defines swarm name and swarm PIN/password. User should know swarm PIN/password to join the desired swarm. Users could make and join to more than one swarm.
